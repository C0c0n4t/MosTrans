Требования: Python 3.9, ffmpeg и библиотеки из requirements.txt (pip install -r requirements.txt)

Запуск: main.py.
Запуск сервера может занять некоторое время, так как для корректной работы понадобятся словари для автокоррекции и транскрибирования русского языка,
которые будут скачиваться при первом запуске.

После запуска в терминале появится ссылка на локальный сайт (http://127.0.0.1:8080).
Перейдя на сайт, можно пользоваться всеми его функциями (пока активен процесс питон и, соответственно, сам сервер).
